## Understanding Git Workflow
Explain in 2–3 sentences the difference between:
- Local workspace (Codespace)
- Remote repository (GitHub.com)
Codespace, the local workspace, creates a temporary development environment that is cloud-based. It is connected to a GitHub repository, but the coding or changes done in Codespace are not added to the repository unless you commit and sync them. This is where you can work on code without worrying about breaking anything. The remote repository, hosted on GitHub, is kind of like a library created for a your code where you can store it and track changes. This is where you save final versions and host your code for others to find.
