Megan Larson

Specializing in Spatial Technologies & Urban Challenges

I am a Master’s candidate in Geographic Information Systems Technology at the University of Arizona. I specialize in the application of spatial technologies to solve complex urban and environmental challenges. My experience involves managing the entire spatial data lifecycle—from acquiring and processing remote sensing imagery to designing interactive Web GIS applications for stakeholder engagement.

I am passionate about the art and science of cartography, ensuring that every analysis is supported by precise, professional, and intuitive visual design. This portfolio highlights my work in Open Source GIS, demonstrating how free, professional tools can solve real-world mapping problems.

Technical Capabilities

Portfolio Projects

A collection of work from GIST 604B, showcasing a journey from basic map-making to automated spatial workflows and full-stack web applications.

WebGIS & Full-Stack Orchestration

Built a live, interactive web map for Nampa, Idaho parks. Users can view boundaries, find amenities, and trace pathway routes in real-time using Leaflet and open-source web technologies.

OSM & PostGIS Spatial Analysis

Established an OSM-to-PostGIS workflow to analyze urban data. I adapted SQL queries to a new area of interest, creating custom spatial visualizations and insights within a notebook environment.

PostGIS Database Orchestration

Configured a PostGIS environment to manage large New York City datasets. Developed complex SQL queries to extract spatial insights from urban infrastructure and census data.

Python GIS & Containerization

Developed Python functions for spatial analysis using pandas, GeoPandas, and Rasterio. Managed the environment using GitHub Codespaces for consistent, cloud-based processing.

QGIS Desktop GIS

A comprehensive foundation in desktop GIS, covering coordinate reference systems (CRS), attribute management, vector/raster styling, and professional cartographic map production.
