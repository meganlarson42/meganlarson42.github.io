# Megan Larson – GIS Portfolio

## About
I am a student in my final year of the MS-GIST program at the University of Arizona. I completed my BS in Geoscience at Boise State University in 2021. This portfolio contains all the work I completed in my “GIST 604B - Open Source GIS” course. The projects demonstrate how to use free, open-source tools to solve real-world mapping problems, from analyzing city parks in Idaho to managing urban data in New York. It shows my journey from basic map-making to building automated scripts and interactive web maps that anyone can use in a browser.

## Skills Demonstrated
- Python: Used to automate GIS tasks and process data with libraries like pandas and GeoPandas.
- QGIS: Employed as a desktop platform for cartography, data management, and spatial analysis.
- PostGIS & PostgreSQL: Utilized to build and query powerful spatial databases for large datasets.
- Leaflet: Used to create interactive, web-based maps with HTML, CSS, and JavaScript.
- GitHub & Codespaces: Provided a cloud-based environment for coding and hosting project repositories.
- Docker: Managed consistent database environments using containerization.
- Rasterio: Enabled the processing and analysis of satellite imagery and raster datasets.
- OpenStreetMap (OSM): Served as a primary source for real-world geographic data across multiple projects.
- SQL: Used to write spatial queries for analyzing relationships between geographic features.
- Git: Facilitated version control and the deployment of web maps through GitHub Page
